List - Lista – Zestaw powiązanych ze sobą obiektów. Listy tworzymy stosując nawiasy prostokątne, a zmienne oddzielamy przecinkami. 


Spójrz na przykład : 

#Lista zwierzaków
zwierzaki = [kot, zebra, koń, pies, ryba]

#Temperatura obserwowana przez cały tydzień
temp = [0, -1, -1, 2, -4, -5, -3]

#Obiekty tekstowe
words = ['Witaj', 'na', 'naszym', 'kursie']

#Rożne obiekty
mobile_phone_details = ["apple", "iphone11", '64GB', 'Space Grey']

#Lista w liście
mobile_phones = [ ["apple", "iphone11", '64GB', 'Space Grey'], ['Samsung', 'Galaxy Note 5', '128GB' 'Grey']]


Listy można dostosować do własnych potrzeb. Możemy je powiększać o kolejne elementy, a także usuwać niepotrzebne. Do takich działań wykorzystujemy metody: 

  • Append
  • Insert
  • Extend
  • Remove
  • Pop 


Append 

Numbers = [1, 2, 3, 4, 5]

Numbers.append(5)


Print(Numbers)

Metoda ta dodaje pojedynczą wartość na końcu listy.

Sprawdź poniższy kod : 

Numbers = [1, 2, 3, 4, 5]

Numbers.insert(1,3)

Print(Numbers)


Metoda insert przyjmuje dwa argumenty. W pierwszym podajemy numer indeksu, który przyjmie nasza dodana wartość. Drugim argumentem jest wybrana przez nas wartość. 



Extend 

Sprawdź poniższy kod : 

numbers = [1, 2, 3, 4, 5]

numbers.extend([6,7,8,9])

print(numbers)

Metoda insert pozwala dodawać kolejne listy do naszej listy. 


Remove 

Sprawdź poniższy kod : 

numbers = [1, 2, 3, 4, 5]

numbers.remove(3)

print(numbers)

Metoda remove pozwala na usunięcie obiektów. Jej argumentem jest wartość wybranego obiektu, która zostaje znaleziona na liście i usunięta. 


Pop 

numbers = [1, 2, 3, 4, 5]

numbers.remove(3)

print(numbers)


Dzięki metodzie pop można usunąć obiekty z listy używając indeksów. W przypadku niepodania indeksu numbers.pop() python usunie ostatni element z listy.
